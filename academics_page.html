EECS 388 - Introduction to Computer Security

Python:

  • A length extension and hash collision attack on MD5 hashes.
  • A Vigenere deciphering tool that uses variance in ciphertext letter frequency to deduce the key length,
    and uses chi-square test to decode the individual key at each index of the whole key.
  • A Bleichenbacher’s attack for the RSA signature forgery.
  • A Padding Oracle attack against cipher block chaining.
  • Buffer Flow attack that utilizes Return to libc, Integer Overflow and Return Oriented Programming
    to open a shellcode on C programs with DEP and ASLR defenses.

Go:

  • A pcap file analyisis tool that catches devices attempting port scanning or ARP spoofing attacks.
  • A man-in-the-middle (mitm) attack on a DNS server to trick the user into continuously sending requests to a malicious server.

Javascript & JQuery:

  • SQL injection against a website that utilizes unsanitized user inputs to log in as a user whose password is unknown.
  • XSS attack to steal user information to a remote server from a vulnerable website with increasing level of input sanitization.
  • CSRF attack against a website with token validation or XSS defence to log the victim in as a malicious user
    which the attacker has control over.

Others:

  • Analyisis of vulnerabilities within a pcap file's network traffic using Wireshark.
  • Digital forensic to perform live and dead analysis using Autopsy, Virtual Machine and other tools.
